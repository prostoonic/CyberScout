Уровень 5. Разрешения приложений

Приложения могут запрашивать доступ к камере, микрофону, контактам и другим данным. Не все разрешения безопасны.

Если дать лишний доступ, приложение может следить за тобой или передавать твои данные.

Как понять, можно ли дать разрешение:

* Оно должно быть нужно для работы приложения
* Если запрос выглядит странно — лучше отказать
* Не давай лишние разрешения
* При необходимости выбирай доступ только во время использования

Примеры:
Фонарик просит доступ к контактам — подозрительно
Навигатор просит геолокацию — нормально
Соцсеть просит камеру — допустимо

Главное правило: давай только необходимые разрешения.

Вот описание для уровней
1. Навигатор
Это приложение для построения маршрутов и поиска мест поблизости.

Необходимые разрешения:

Геолокация (для определения местоположения на карте).

Микрофон (для голосового поиска адреса, чтобы не отвлекаться от дороги).

2. Фоторедактор
Инструмент для наложения фильтров и ретуши уже готовых снимков.

Необходимые разрешения:

Галерея (чтобы выбрать фото для редактирования и сохранить результат).

Камера (чтобы сделать снимок сразу внутри приложения).

3. Мессенджер
Приложение для текстовых сообщений, звонков и обмена файлами.

Необходимые разрешения:

Контакты (чтобы найти друзей, которые уже пользуются сервисом).

Микрофон (для аудиозвонков и записи голосовых сообщений).

Галерея (для отправки картинок и видео).

4. Видеочат
Платформа исключительно для проведения видеоконференций.

Необходимые разрешения:

Камера (чтобы собеседники вас видели).

Микрофон (чтобы вас слышали).

5. Прогноз погоды
Простой виджет, показывающий температуру и осадки.

Необходимые разрешения:

Геолокация (чтобы показывать прогноз именно для вашего города в автоматическом режиме).
